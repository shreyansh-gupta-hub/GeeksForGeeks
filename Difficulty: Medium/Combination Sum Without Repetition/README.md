<h2><a href="https://www.geeksforgeeks.org/problems/combination-sum-ii-1664263832/1">Combination Sum Without Repetition</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO"><p><span style="font-size: 18px;">Given an array&nbsp;<strong>arr[]</strong>&nbsp;and a&nbsp;<strong>target</strong>, your task is to find all&nbsp;<strong>unique&nbsp;</strong>combinations in the array where the sum of the elements is equal to target. </span><span style="font-size: 18px;">Each element in arr[] can be used at most <strong>once </strong>in the combination.</span></p>
<p><span style="font-size: 18px;"><strong>Note:&nbsp;</strong>You can return your answer in&nbsp;any<strong>&nbsp;</strong>order, the driver code will print them in sorted order.<br>Test cases are generated such that number of unique combination does not exceed 10<sup>6</sup>.</span></p>
<p><strong><span style="font-size: 18px;">Examples:</span></strong></p>
<pre><span style="font-size: 18px;"><strong>Input:</strong> </span><span style="font-size: 18px;">arr[] = [1, 2, 3], target = 5</span>
<strong><span style="font-size: 18px;">Output: </span></strong><span style="font-size: 18px;">[[2, 3]]</span>
<strong><span style="font-size: 18px;">Explanation: </span></strong><span style="font-size: 14pt;">There is only one unique combinations whose sum is equal to the target.</span></pre>
<pre><span style="font-size: 18px;"><strong><span style="font-size: 18px;">Input:</span> </strong></span><span style="font-size: 18px;">arr[] = [1, 3, 2, 2, 2], target = 4</span>

<strong><span style="font-size: 18px;">Explanation: </span></strong><span style="font-size: 14pt;">There are two unique possible combinations whose sum is equal to target</span>.</pre>
